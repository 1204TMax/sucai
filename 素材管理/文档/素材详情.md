# 素材详情需求文档

## 1. 页面基本信息
- **页面名称**：素材详情
- **所属模块**：素材管理
- **文件路径**：`素材管理/素材详情.html`
- **页面定位**：用于承载单个素材的完整信息查看，包括预览、基础信息、数据总览与媒体信息。

## 2. 进入方式
- 从 `素材管理/素材库.html` 点击素材卡片主体区域时，在系统框架中打开本页新标签页。
- 从 `素材管理/素材库.html` 点击素材卡片底部 `编辑` icon 时，也打开本页新标签页。
- 两个入口需统一进入同一详情页，不拆分“查看详情”和“编辑详情”两套页面。

## 3. 页面结构
### 3.1 顶部区域
- 页面顶部展示“素材详情”标题。
- 顶部保留关闭、编辑等操作入口。

### 3.2 首屏布局
- 首屏采用左右两栏布局：左侧为素材预览，右侧为基础信息。
- 支持图片 / 视频两类素材展示。

### 3.3 信息分区
- 基础信息
- 数据总览
- 媒体信息（按平台切换）
- 其他补充信息

## 4. 交互要求
- 页面通过系统框架新标签页打开，不以内嵌抽屉展示。
- 页面内容允许纵向滚动，但应保留清晰的信息分区与标题层级。
- 返回素材库时不应影响素材库当前的筛选、排序和分页上下文。
