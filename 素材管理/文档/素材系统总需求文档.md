# 素材系统总需求文档

## 1. 文档信息

| 项目 | 内容 |
|---|---|
| 文档名称 | 素材系统总需求文档 |
| 所属项目 | 量化投放系统 / 素材管理 |
| 文档目标 | 统一当前阶段素材系统的业务范围、页面范围、核心对象、主流程、状态口径与边界定义 |
| 当前范围 | 仅覆盖素材管理当前已明确范围：素材库、物料库、素材/物料上传、素材详情、合成策略、新建策略、我的任务 |
| 不含范围 | 不展开媒体审核系统、投放系统、效果分析平台、权限系统、外部平台接口细节 |
| 文档日期 | 2026-04-21 |

## 2. 背景与目标

| 维度 | 说明 |
|---|---|
| 业务背景 | 当前业务需要对图片、视频素材及视频片段物料进行统一管理，并支持上传、分类、检索、下载、合成生产与任务追踪。 |
| 当前问题 | 素材、物料、合成任务与上传下载任务分散，页面口径不统一，难以形成完整的生产与管理闭环。 |
| 建设目标 | 建立一套面向素材生产与管理的工作台，统一管理素材、物料、合成策略及相关异步任务。 |
| 本阶段目标 | 先完成当前范围内页面、流程、状态与对象定义，为后续原型、页面开发、接口设计提供统一依据。 |

## 3. 术语定义

| 术语 | 定义 |
|---|---|
| 素材 | 可直接用于媒体投放的图片或视频资源。 |
| 物料 | 用于剪辑、拼接、合成素材的视频片段或图片片段。 |
| 合成 | 按既定规则将多个物料拼接或组合生成素材的过程。 |
| 合成策略 | 对物料组合规则、筛选条件、生成数量、定向范围等进行定义的业务配置。 |
| 上传包 | 一次批量上传中的任务容器，可包含多个单文件上传任务。 |
| 合成包 | 一次视频合成任务中的任务容器，可包含多个结果文件。 |
| 任务中心 | 素材管理模块内统一查看上传、下载、视频合成任务的页面。 |

## 5. 页面需求总览

| 页面 | 页面类型 | 核心职责 | 上游输入 | 下游输出 |
|---|---|---|---|---|
| 素材库 | 工作台页 | 查看、检索、分类管理素材，发起上传、下载、移动、收藏、入库等操作 | 已入库素材、筛选条件 | 素材详情、下载任务、上传素材页 |
| 物料库 | 工作台页 | 查看、检索、分类管理物料，发起上传与批量处理 | 已入库物料、筛选条件 | 上传物料页、后续合成使用 |
| 合成策略 | 工作台页 | 查看、筛选、管理合成策略，监控策略状态与产出规模 | 已创建策略 | 新建策略页、视频合成任务 |
| 我的任务 | 工作台页 | 统一查看上传、下载、视频合成任务及处理结果 | 各业务页产生的异步任务 | 任务详情、结果查看 |
| 素材详情 | 业务页 | 查看单个素材的基础信息、数据总览与媒体信息 | 素材库中的素材对象 | 返回素材库上下文 |
| 上传素材 | 业务页 | 完成素材上传、检测、审核、入库前处理 | 本地文件、目标目录、补充信息 | 上传任务、待入库素材 |
| 上传物料 | 业务页 | 完成物料上传、检测、审核、入库前处理 | 本地文件、目标目录、补充信息 | 上传任务、待入库物料 |
| 新建策略 | 业务页 | 配置合成规则、节点筛选条件和目标数量 | 项目、定向、标签、物料范围 | 合成策略 |

## 6. 核心业务对象

| 对象 | 定义 | 当前阶段核心用途 |
|---|---|---|
| 素材 | 可直接投放的图片/视频 | 在素材库中管理、查看详情、下载、分类与数据分析 |
| 物料 | 合成上游输入资源 | 在物料库中管理，并作为合成策略筛选对象 |
| 合成策略 | 规则配置对象 | 定义物料选择规则、节点结构、生成数量与定向范围 |
| 上传任务 | 异步处理对象 | 跟踪素材/物料上传、检测、审核、入库过程 |
| 下载任务 | 异步处理对象 | 跟踪素材/物料下载过程 |
| 视频合成任务 | 异步处理对象 | 跟踪按策略生成结果素材的过程 |

## 7. 核心流程

| 流程 | 起点页面 | 关键环节 | 终点/结果 |
|---|---|---|---|
| 素材上传 | 素材库 | 选择目录 → 上传文件 → 同质化检测 → 审核 → 待入库 → 手动入库 | 素材进入素材库；过程沉淀到我的任务/上传 |
| 物料上传 | 物料库 | 选择目录 → 上传文件 → 同质化检测 → 审核 → 待入库 → 手动入库 | 物料进入物料库；过程沉淀到我的任务/上传 |
| 素材下载 | 素材库 | 发起下载 → 创建下载任务 → 下载执行 | 结果沉淀到我的任务/下载 |
| 策略创建 | 合成策略 | 新增策略 → 配置全局参数 → 配置节点筛选条件 → 保存策略 | 生成可用的合成策略 |
| 策略编辑 | 合成策略 | 打开既有策略 → 修改配置 → 保存 | 更新既有策略 |
| 视频合成 | 合成策略 | 策略生效/执行 → 生成合成包任务 → 结果输出 | 结果沉淀到我的任务/视频合成 |
| 素材查看 | 素材库 | 打开素材详情 | 查看素材基础信息、数据总览、媒体信息 |

## 8. 状态口径

### 8.1 上传任务状态

| 状态 | 说明 |
|---|---|
| 待执行 | 任务已创建，尚未开始处理 |
| 上传中 | 文件正在上传 |
| 上传成功 | 文件已上传完成 |
| 同质化检测中 | 正在执行同质化检测 |
| 检测未通过 | 同质化检测未通过 |
| 待审核 | 等待审核 |
| 审核不通过 | 审核未通过 |
| 待入库 | 审核通过，等待用户确认入库 |
| 入库中 | 正在入库 |
| 入库失败 | 入库失败 |
| 已完成 | 已完成上传与入库 |
| 已删除 | 任务已删除 |

### 8.2 下载任务状态

| 状态 | 说明 |
|---|---|
| 下载中 | 正在下载 |
| 下载失败 | 下载失败 |
| 下载成功 | 下载完成 |

### 8.3 视频合成任务状态

| 状态 | 说明 |
|---|---|
| 等待中 | 已进入队列，等待执行 |
| 已暂停 | 任务被人工暂停 |
| 同质化检测中 | 正在执行同质化检测 |
| 同质化合格 | 检测通过 |
| 同质化过高 | 检测未通过，结果相似度过高 |
| 上传中 | 合成结果正在上传 |
| 上传成功 | 结果上传完成 |
| 上传失败 | 结果上传失败 |
| 已删除 | 任务已删除 |

### 8.4 合成策略状态

| 状态 | 说明 |
|---|---|
| 生效中 | 策略可继续生产任务 |
| 已暂停 | 策略已暂停，不再继续生产 |

## 10. 数据要求

### 10.1 素材/物料字段分层

| 字段层 | 内容 |
|---|---|
| 基础信息 | ID、名称、MD5、类型、格式、尺寸、比例、大小、时长、封面等 |
| 来源信息 | 素材来源、获取方式 |
| 时间信息 | 上传时间、创建时间、修改时间 |
| 标签信息 | 类型标签、内容标签、AI标签、定向标签、数据标签 |
| 生产信息 | 设计师、剪辑、编导、策划、拍摄、演员、供应商等 |
| 状态信息 | 使用状态、新旧状态、效果评估状态、风险评估状态、审核状态 |
| 媒体信息 | 媒体平台、媒体素材ID、审核信息、评论反馈 |
| 投放信息 | 广告账户、广告计划、广告ID、创意ID、投放模式、系统平台 |
| 效果信息 | 展示、点击、播放、互动、转化、ROI、留存等指标 |
| 治理信息 | 操作日志、上传记录、修改记录、批注、可见范围、管理权限 |

### 10.2 当前阶段页面需要重点承接的数据对象

## 13. 业务规则与边界

| 规则 | 说明 |
|---|---|
| 菜单边界 | 素材管理侧边栏固定为：素材库、物料库、合成策略、我的任务。 |
| 页面承载边界 | 上传、详情、策略编辑等非固定菜单页面，不直接加入侧边栏。 |
| 任务归集规则 | 上传、下载、视频合成三类异步任务统一沉淀到“我的任务”。 |
| 入库规则 | 上传成功后需经过检测与审核，审核通过后进入“待入库”，由用户手动确认入库。 |
| 策略规则 | 合成策略支持新建、编辑、暂停、启动；暂停后不继续生产新任务。 |
| 查看规则 | 素材详情为独立业务页，用于查看单个素材完整信息。 |
| 当前文档边界 | 本文档定义业务范围与口径；页面字段与按钮已补充到产品需求粒度，视觉样式与组件行为另文说明。 |
