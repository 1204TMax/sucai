# 物料库需求文档

## 1. 页面基本信息
- **页面名称**：物料库
- **所属模块**：素材管理
- **文件路径**：`素材管理/物料库.html`
- **页面定位**：用于管理物料资产，支持检索、分类查看、上传与批量操作。

## 2. 页面结构
页面采用左侧分类树 + 右侧物料网格的双栏布局。

### 2.1 顶部操作区
- 顶部保留查询、清空等基础筛选能力。
- 提供 `上传物料` 主按钮，点击后需在系统框架内打开 `素材管理/上传物料.html` 新标签页。
- 同步物料库、新建文件夹、删除、全选等入口保留在当前页面内处理。

### 2.2 左侧分类树
- 展示物料分类层级。
- 支持默认展开当前激活目录，并与右侧内容区域联动。

### 2.3 右侧内容区
- 采用网格方式展示物料卡片。
- 支持排序与列表 / 网格视图切换。

## 3. 交互要求
- 所有进入独立业务页的入口需通过系统框架 `openTab` 打开，不直接替换当前 iframe。
- 当前已接入的独立页面入口为：`上传物料` → `素材管理/上传物料.html`。
